题目：

1.“+”运算符的重载，要求两个长方形相加时，长与长相加，宽与宽相加。

2.流提取运算符“>>”做重载时，分别对长方体对象的长和宽输入数值，输入前，要有文字提示。

3.流插入运算符“<<”做重载时，要输出长方形的长、宽、面积。

4.在main函数中建立3个Rectangle的对象，实现三个符号的功能，最终输出3个对象的所有信息。

5.程序输出的形式参考下面的实验范例结果，但数据要与其不同。

实验范例结果：
